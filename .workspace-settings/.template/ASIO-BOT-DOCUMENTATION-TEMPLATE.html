Step-by-Step Execution Guide

General Execution Steps

  1. Open ConnectWise PSA Ticket
    • Navigate to the client ticket via ConnectWise PSA (Manage)
    • Note the ticket number
  2. Launch ASIO Bot
    • Access Automations Tab
    • Search for: {Bot Name}
    • Click to open bot execution form
  3. Fill Required Inputs
    • Ticket Number: Enter ticket number
    • Operation: Select from dropdown
    • {Other Inputs}: Fill as required
  4. Validate Inputs
    • Ensure all required fields are filled
    • Ensure any optional fields are filled
    • Verify data accuracy
  5. Execute Bot
    • Click Run (or CTRL+SHIFT)
    • Wait for completion. (Cloud bots 30sec-1min. Local bots 1-2min.)
  6. Review Results
    • Check execution logs for SUCCESS/WARNING/FAILED messages
    • Verify expected outcomes
