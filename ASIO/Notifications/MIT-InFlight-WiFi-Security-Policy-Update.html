InFlight WiFi Security Access update
Important: Accessing Microsoft Services via In-Flight WiFi
We want to inform you about a recent issue affecting access to Microsoft services, including email, when using in-flight WiFi.

Background:

Our security baseline blocks access to Microsoft services from locations outside Australia.
Previously, while using QANTAS onboard WiFi services, users were identified as being located within Australia.
Recent changes in their system mean that in-flight WiFi connections now show users as being located in the United States, causing connection issues for users who are travelling.

What is Changing:

Mangano IT will implement a new security policy allowing access to Microsoft services when users are identified as sourcing from a known aircraft WiFi service.

  • Users will be prompted to provide MultiFactor Authentication to access services.
  • The change applies only while accessing services from the aircraft.
  • International travelers must still request access for the countries and dates they plan to visit.
  • There may be instances where a user is unexpectedly blocked from access due to the ongoing nature of network address changes on the internet.

No additional costs:

There will be no charge for updating your security configuration.

Important Timeline:

If we do not hear from you, we will roll out this change to your tenancy during the first week of December 2025.
